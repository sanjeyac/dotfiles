# Install

clone the repository somewhere

`git clone https://github.com/sanjeyac/dotfiles.git`

`cd dotfiles`

Replace bash with Zsh + OhMyZsh

`./dotfiles.sh zsh`

Install all third-party tools

`./dotfiles.sh tools`

Install DotFiles

`./dotfiles.sh install`

Solarized theme

`./dotfiles.sh solarized`
